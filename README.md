# Car Dealership Review and Rating Application

## Project Overview
The Car Dealership Review and Rating Application is a full-stack web application developed as part of the capstone project. The application allows users to browse car dealerships, view dealer details, read customer reviews, and submit reviews after authentication.

The project is built using Django, React.js, Node.js, MongoDB, Bootstrap, and IBM Cloud services.

--------------------------------------------------

## Features

- User authentication and login
- Browse car dealerships
- View dealership details
- Add and view reviews
- Sentiment analysis for reviews
- Responsive frontend design
- REST API integration
- Cloud deployment support

## Installation Guide

Clone Repository:

git clone https://github.com/arslangrt26/cars-dealership-capstone.git

cd cars-dealership-capstone

--------------------------------------------------

Create Virtual Environment:

python -m venv env

--------------------------------------------------

Activate Environment

Windows:

env\Scripts\activate

Linux/Mac:

source env/bin/activate

--------------------------------------------------

Install Dependencies:

pip install -r requirements.txt

--------------------------------------------------

Frontend Setup:

cd frontend

npm install

npm start

--------------------------------------------------

Run Django Server:

python manage.py runserver

--------------------------------------------------
